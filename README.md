# Global Superstore Dashboard

This **Power BI dashboard** provides a comprehensive overview of the Global Superstore's operations, enabling data-driven decisions on sales performance, product trends, and regional insights.

## Key Features

1. **Sales Overview**: Total sales, average order value, and profit margins with dynamic KPI cards  
2. **Sales by Region & Category**: Interactive maps and bar charts to identify top-performing regions and product categories  
3. **Customer Insights**: Track order trends, customer segments, and repeat purchases  
4. **Profit & Loss Analysis**: Identify high-margin products and areas needing attention  
5. **Time Trends**: Monthly and quarterly sales trends to spot seasonal patterns  
6. **Slicers & Filters**: Filter by region, category, segment, and order date for interactive analysis  

## Business Insights

- Top-selling product categories differ across regions  
- Profit margins vary significantly by segment and product line  
- Certain regions have strong sales volume but low profitability  
- Seasonal trends impact sales; peak months identified for marketing focus  

## Tech & Tools

- Power BI Desktop  
- DAX for KPI calculations (Total Sales, Profit %, Average Order Value)  
- Interactive visuals for operational and strategic insights  
- Clean, business-friendly dashboard design  

## Notes

- Dataset: Simulated Global Superstore dataset (for practice and portfolio purposes)  
- Dashboard is fully interactive; slicers and filters demonstrate typical retail analytics
